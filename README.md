# Sistema Inteligente de Peaje Automatizado

Proyecto académico orientado al diseño y desarrollo de una plataforma inteligente para la gestión de peajes, pagos digitales, monitoreo vehicular y generación de alertas por vehículos reportados.

## Tecnologías principales

- Django
- Django REST Framework
- PostgreSQL
- Docker
- Docker Compose
- React
- Flutter

## Módulos previstos

- Usuarios y autenticación
- Gestión de vehículos
- Peajes y cámaras
- Billetera virtual
- Transacciones
- Membresías
- Aviso interno de vehículo robado
- Alertas de seguridad
- Ubicación de detección
- Notificaciones
- Historial de usuario
